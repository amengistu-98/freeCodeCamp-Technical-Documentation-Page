Introduction

Python is a high-level, general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation. Python is dynamically typed and garbage-collected. It supports multiple programming paradigms, including structured, object-oriented and functional programming.

Guido van Rossum began working on Python in the late 1980s as a successor to the ABC programming language and first released it in 1991 as Python 0.9.0. Python 2.0 was released in 2000. Python 3.0, released in 2008, was a major revision not completely backward-compatible with earlier versions. Python 2.7.18, released in 2020, was the last release of Python 2.

Python consistently ranks as one of the most popular programming languages.

Variables

A Python variable is a symbolic name that is a reference or pointer to an object. Once an object is assigned to a variable, you can refer to the object by that name.

Rules for Python variables:

  • • A variable name must start with a letter or the underscore character
  • • A variable name cannot start with a number
  • • A variable name can only contain alpha-numeric characters and underscores (A-z, 0-9, and _ )
  • • Variable names are case-sensitive (age, Age and AGE are three different variables)
#legal variable names: myvar = "John" my_var = "John" _my_var = "John" myVar = "John" MYVAR = "John" myvar2 = "John" #illegal variable names: 2myvar = "John" my-var = "John" my var = "John"
Data Types

Variables can store data of different types. Different types of data do different things.

Data types in Python belong to these categories:

  • • Text Types: str
  • • Numeric Types: int, float, complex
  • • Sequence Types: list, tuple, range
  • • Boolean Types: bool
  • • Binary Types: bytes, bytearray, memoryview
  • • None Types: NoneType

To print the data type of the variable x

x = 5 print(type(x))
Statements

Python supports the usual logical conditions from mathematics.

  • • Equals: a == b
  • • Not Equals: a != b
  • • Less than: a < b
  • • Less than or equal to: a <= b
  • • Greater than: a > b
  • • Greater than or equal to: a >= b

These conditions can be used in different ways, such as in if statements, while loops, and for loops:

if statements

The if statement executes one set of code if a specified condition is met (true) or another set of code evaluates to false.

#if statement: a = 10 b = 15 if b > a<   print("b is greater than a")

In the if statement example, there are two variables, a and b. The if statement tests to see whether b is greater than a. Since a is 10 and b is 15, the output will be:

b is greater than a

If the variables values were switched (a = 15 and b = 10), then nothing will be printed since the conditions of the if statement is not met.

while loops

With the while loop we can execute a set of statements as long as a condition is true.

#while loop: i = 1 while i < 6   print(i)   i += 1

In the while loop example, there is a variable i, which is set to 1. while i is less than 6, the while loop will print the value of i, and then incrememnt i by 1. The loop will repeat until i is no longer less than 6. This means that the output of the above statement will be:

1 2 3 4 5 6

Note: remember to increment i, or else the loop will continue forever!

for loops

A for loop is used for iterating over a sequence (that is either a list, a tuple, a dictionary, a set, or a string).

#for loop: fruits = ["guava", "orange", "banana"] for x in fruits:   print(x)

In the while loop example, we have a list of fruits. The for loop goes through the fruits in the list, and then prints them out. The output of this statement will be:

guava orange banana

Note: that the print function within the if statement, for loop, and while loop is indented. If it were not indented, an error will be raised. In other words, if statements require indentation!
